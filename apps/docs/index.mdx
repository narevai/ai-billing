# AI Billing

`ai-billing` is middleware for the [Vercel AI SDK](https://sdk.vercel.ai/docs) that captures usage and sends normalized billing events to external billing systems.

It helps you:

- calculate usage costs with provider-aware middleware
- add request-level tags (such as user or org metadata)
- forward events to destinations like Polar, Stripe, OpenMeter, and Lago

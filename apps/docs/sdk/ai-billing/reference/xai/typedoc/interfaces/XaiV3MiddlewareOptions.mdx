Configuration for [createXaiV3Middleware](/sdk/ai-billing/reference/xai/typedoc/functions/createXaiV3Middleware).

Extends BaseBillingMiddlewareOptions (`destinations`, `defaultTags`, `waitUntil`, `onError`) and
requires a PriceResolver. Usage is taken from the AI SDK's normalized usage fields; cost is
computed from that usage and the resolved ModelPricing using the same rules as the package's cost
helper.

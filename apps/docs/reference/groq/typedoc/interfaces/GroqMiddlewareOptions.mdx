Configuration for [createGroqV3Middleware](../functions/createGroqV3Middleware).

Extends BaseBillingMiddlewareOptions (`destinations`, `defaultTags`, `waitUntil`, `onError`) and
requires a PriceResolver. Usage is taken from the Groq response; cost is computed from that usage
and the resolved ModelPricing using the same rules as the package's cost helper.

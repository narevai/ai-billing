# `@ai-billing/core`

`@ai-billing/core` is the foundational package for AI Billing.

It provides shared billing primitives:

- canonical types like `BillingEvent`, `Usage`, `Cost`, and `ModelPricing`
- generic middleware helpers for AI SDK integrations
- destination helpers and metadata mapping utilities
- pricing resolver and cost conversion utilities
- a shared error hierarchy for billing-related failures

## Recommended usage

Most users should **not** integrate `@ai-billing/core` directly.

Instead, use provider and destination packages (for example `@ai-billing/openai`, `@ai-billing/groq`, `@ai-billing/openrouter`, `@ai-billing/openmeter`) that already compose these primitives for common production use cases.

Reach for `@ai-billing/core` directly when you need custom provider middleware, custom destinations, or advanced billing pipelines.

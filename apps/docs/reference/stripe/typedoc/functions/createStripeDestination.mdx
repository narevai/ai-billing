# Function: createStripeDestination()

> **createStripeDestination**\<`TTags`\>(`options`): `Destination`\<`TTags`\>

Defined in: [stripe-destination.ts:61](https://github.com/narevai/ai-billing/blob/main/packages/stripe/src/destination/stripe-destination.ts#L61)

Creates a Destination that ingests billing events into Stripe Meters.

Identity is extracted from tags using `stripe_customer_id`. If no identity is present, a warning is
logged; the current implementation still sends the event with an `undefined` identity value.

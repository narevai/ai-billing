# @ai-billing/baseten

Baseten billing for the AI Billing SDK. Uses free cache-hit semantics: cached tokens are subtracted from the prompt bucket and are not billed separately.

**Installation**

```bash
pnpm add @ai-billing/core @ai-billing/baseten
```

**Documentation**

For full usage instructions and examples, see the [Documentation](https://www.narev.ai/docs/sdk/ai-billing).
